Detailed Description

PathIOKeyboard — concrete IO provider for keyboard devices.

Notes:

  • This class does not know where it is mounted in a parent PathSpace.
  • It exposes a thread-safe simulated event queue API to feed events from tests or platform backends (macOS, etc). The out()/in() behavior is inherited from PathIO.
  • A future implementation will override out() to deliver KeyboardEvent directly (peek or pop depending on Out options), and use notify(...) to wake waiters.
